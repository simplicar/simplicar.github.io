About the position

We heavily use front-end technologies like AngularJS to develop modern web applications for our users. We are currently doing some research on ReactJS and ES6 to implement a new feature which will make a huge impact on the business.

As a front-end developer you will be responisble for the development of UI components across all of our projects.

Must-have
  • Expert knowledge of JavaScript & AngularJS (or at least one other MVC library / framework like BackboneJS, EmberJS, etc.)
  • Expert knowledge of object-oriented JavaScript (pure JavaScript, DOM API) and design patterns
  • Expert knowledge of HTML, cross-browser CSS
  • Ability to write high-performance, reusable code for UI components
  • Understanding of RESTful application architecture
Nice to have
  • Experience using git
  • Experience with TDD/automated testing, CI and any unit testing framework (Jasmine, Mocha, etc.)
  • Experience with working with Agile \ SCRUM
